# 🎬 YT Downloader

![Python](https://img.shields.io/badge/Python-3.10-blue)
![PyQt6](https://img.shields.io/badge/UI-PyQt6-green)
![Status](https://img.shields.io/badge/status-active-success)

**YT Downloader** — это современное настольное приложение для загрузки видео и аудио с популярных интернет-платформ.
Программа разработана на Python с использованием PyQt6 и библиотеки yt-dlp, что обеспечивает высокую скорость, стабильность и удобный пользовательский интерфейс.

## ✨ Возможности

* 📥 Скачивание видео (MP4)
* 🎵 Скачивание аудио (MP3)
* 🎯 Выбор качества видео
* 🖼 Предпросмотр (превью, название, длительность)
* 📊 Отображение прогресса загрузки
* 📁 Выбор папки сохранения
* ⚡ Быстрый и нормальный интерфейс

## 🛠 Технологии

* Python 3
* PyQt6 (GUI)
* yt-dlp (скачивание видео)
* Pillow (работа с изображениями)

## 💪 Умные функции
 * Автоматическое получение информации о видео
 * Оценка размера файла до загрузки
 * Повторные попытки при ошибках
 * Поддержка cookies (при необходимости)

## ✨ Дополнительные возможности
·	Выбор папки для сохранения
·	Отмена загрузки в любой момент
·	Отображение имени скачиваемого файла


## 🚀 Установка

```bash
git clone https://github.com/Anyusek/YT_downloader.git
cd yt-downloader
pip install -r requirements.txt
python yt_downloader
```

## ⚠️ Важно

* Для скачивания аудио требуется установленный **FFmpeg**
* Некоторые видео могут требовать `cookies.txt`
* DRM-защищённые видео скачать нельзя
